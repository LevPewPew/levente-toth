Makeup by Ellie

This website by working with client requirements to bring them a website that could show off her work portfolio, inform customers about their services, and provide a dynamic contact form that would streamline a previously time consuming process of collecting a customers information and needs.

The business own is also provided with Content Management System (CMS) hidden behind authorized access. This CMS allows the business owner to create, update and remove content (services and prices, FAQ, and portfolio works) as desired without the need to have any technical or coding knowledge. It also provides a central area to view contact info of any submitted contact forms in addition to the email originally sent by the form.

This website was the fourth and final coding assignment from the Coder Academy bootcamp. It was completed as a group project using Agile project management and git workflow methodologies.

Grid Mage

I created this website as a way to continue practicing using JavaScript and the DOM. Grid Mage is a result of my interest in Procedural Algorithms, I thought what better way to start learning than attempting to make one myself.

The premise of Grid Mage is to randomly generate rectangular tiles of varying dimensions and align them with no gaps into a CSS grid.

Art Haus

This web app allows users to browse listings of art work by other registered accounts. Signed up users can post listings to sell, and buy the art works of others.

Art Haus features a PostgresQL database, Authorization and Authentication, Amazon S3 cloud storage, Admin functionality, payment processing and much more.

This website was the third coding assignment from the Coder Academy bootcamp.

Objectmon

A simple, very basic pocket monster style game created for the terminal using Ruby. Ascii graphics represent the world map. Combat is text based. Several different types of Objectmon can be chosen to fight for you and to fight against. A persistent high score system using CSV files.

This website was the very first coding assignment from the Coder Academy bootcamp.
